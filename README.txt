========================================
Word Warrior - 自訂單字庫使用說明
========================================

【功能說明】
Word Warrior 支援自訂單字庫功能。遊戲啟動時，會自動從 words.txt 
讀取單字。如果沒有找到 words.txt，則使用內建的預設單字。

【文件格式】
- 檔案名稱：words.txt
- 編碼：UTF-8（使用記事本編輯時，另存新檔並選擇 UTF-8 編碼）
- 每行一個單字（英文 + 空格 + 中文）

【範例格式】
adventure 冒險
challenge 挑戰
dragon 龍
magic 魔法
treasure 寶藏

【如何編輯】
1. 在遊戲同一個目錄中找到 words.txt 檔案
2. 用記事本（或任何文字編輯器）打開
3. 按照「英文 空格 中文」的格式，每行新增一個單字
4. 存檔後，重新執行遊戲即可載入新單字

【注意事項】
✓ 至少要有 3 個單字遊戲才能開始
✓ 單字數量建議 10-50 個，避免過多導致遊戲過於困難
✓ 中文名稱不要包含空格（如需多字，用「」連接）
✓ 不要使用特殊符號（只用英文、中文、空格）
✓ 英文和中文之間必須用一個空格分隔

【範例】
正確：apple 蘋果
正確：fireworks 煙火
錯誤：apple=蘋果（不要用等號）
錯誤：apple蘋果（要用空格隔開）
